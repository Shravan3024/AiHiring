## 🔬 Key Intelligence Features

### 1. Hybrid AI-ML Assessment Engine
Bypassing traditional database limitations, the engine utilizes **JSON Persistence** for 100% data durability.
- **Dual-Path Scoring**: Validates answers via Gemini AI with an automatic failover to **Cosine Similarity Local ML** during network outages.
- **Normalization**: Applies 80/20 weights to Technical/Behavioral scores for precise candidate profiling.

### 2. Proctoring Vanguard (Security Shield)
Enforces a sterile testing environment through:
- **Isolation Breach Detection**: Hard-locks the interface on tab-switches or fullscreen exits.
- **Malpractice Telemetry**: Logs biometric anomalies, copy-paste attempts, and window blurs directly to the secure audit trail.

### 3. Automated Approval Workflows
- **Multi-Stakeholder Concurrence**: Orchestrates decisions across Admin, HR, and MD dashboards.
- **Automated Offer Generation**: Template-based PDF generation triggered upon 3-tier approval logic.

## 🛠️ Deployment Protocol

### 1. Unified Synchronization
The platform is synchronized as a single industrial monorepo.
```bash
git clone https://github.com/Samarth-Ghare/Mask-Polymers
cd Mask-Polymers
```

### 2. Backend Initialization (Port 5000)
```bash
cd backend
npm install
# Sync Sequelize models with Supabase/PostgreSQL
npm run dev
```

### 3. Frontend Activation (Port 3000)
```bash
cd frontend
npm install
npm run dev
```
